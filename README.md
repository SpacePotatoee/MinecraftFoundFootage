# SpacePotato's Found Footage Mod Guide

## Getting Started

### Prerequisites

- Minecraft 1.20.1
- GeckoLib 4.4.9
- Voice Chat 1.20.1-2.4.32

### Adding SP-Backrooms as a Dependency

Add to your `build.gradle`:

```gradle
dependencies {
    modImplementation "com.sp:spb-revamped:${project.spb_version}"
}
```
